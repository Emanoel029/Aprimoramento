  (Modelo de Caixas: primeiros passos) Teoria

HTML CSS box model

 altura= height
 largura= width
 linha q circula o conteúdo = border
 preenchimento interno, da uma descolada = padding
 preenchimento externo, da uma descolada = margin
 contorno fora do elemento = outline

OBS: Sempre que tiver uma caixa do tipo BOX-LEVEL sempre vai começar com uma linha inteira e pula para proxima linha antes de continuar o conteúdo.
     Sempre que tiver uma caixa do tipo INLINE-LEVEL não quebra a linha, começa de onde parou e só ocupa o tamanho que precisar e continua o conteúdo do paragrafo.

      (Grande)
 Ex: Box-level: <div>, <h1>-<h6>, <p>, <main>, <header>, <nav>, <article>, <aside>, <footer>, <from>, <video>
     
     Inline-level: <span>, <a>, <code>, <small>, <strong>, <em>, <sup> - <sub>, <label>, <button>, <input>, <select>
